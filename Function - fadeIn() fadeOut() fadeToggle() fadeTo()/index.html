The primary aim of this project is to search a keyword or line of text within an ebook or multiple ebooks at the same time. It works similar to a search engine like google that searches keyword or text within webpages but in this project i.e. a webapp, we will be able to process ebooks. This project is going to read and search text from ebooks that is in PDF format and return the processed output text to the user. E-books will be categorised in the web application into various sections. User will have choice to select from which ebooks text should be searched. This project will be uploaded on a server so it can be accessible to students or researchers globally. These ebooks are going to stored on the same server. Project backend consist of django which is a python based web framework that executes this project.
